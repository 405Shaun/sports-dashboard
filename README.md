# Greatest Plays in Sports History

A single-page web app of the greatest plays and moments across six sports —
NFL, MLB, NBA, Soccer, NHL, and Golf. Searchable, with favorites and an
in-app video player.

## Live site
https://fanciful-chebakia-eaccba.netlify.app

Publishes automatically from GitHub: https://github.com/405Shaun/sports-dashboard
(push to the `main` branch → Netlify rebuilds in ~30 seconds).

## Files in this folder
- `index.html` — the entire app (HTML, CSS, and JavaScript in one file).
- `manifest.webmanifest` — makes the site installable as a phone app.
- `icon-192.png`, `icon-512.png`, `apple-touch-icon.png`, `favicon.png` — app icons (gold play button on green).
- `qr.png` — plain QR code that opens the live site.
- `share-card.png` — shareable graphic with the QR code + instructions. Text or post this.

## Add it to a phone home screen (the "start button")
- iPhone (Safari): open the link → Share → **Add to Home Screen**.
- Android (Chrome): open the link → menu (⋮) → **Add to Home screen / Install app**.
